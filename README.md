# 培训文件智能自动分类系统

## 项目概述
这是一个为解决企业培训后海量学员文件（如作业、反馈表）人工分类效率低下而开发的自动化工具。它能根据文件名中的学员姓名关键词，**自动、准确、瞬间**地将数千份文件分发到对应的个人文件夹中。

> **核心价值**：将需要数人花费数周的手动归类工作，压缩至**单人几分钟内自动完成**。

## 效果与数据
- **效率提升**：文件分类任务从 **“人工数周”** 降至 **“自动几分钟”**，效率提升**超过99%**。
- **准确率**：基于关键词精确匹配，准确率接近**100%**，杜绝人工操作可能造成的误分、漏分。
- **获得认可**：该方案已运用到相关工作实践中，获得一致好评。

## 技术栈与原理
- **语言**：Python 3
- **核心库**：`os`, `shutil`, `re` (用于文件操作和文本匹配)
- **工作原理**：
  1.  **可配置名单**：用户在脚本开头维护一个简单的学员名单（格式：`英文花名(中文姓名) 小组`）。
  2.  **智能匹配**：脚本扫描当前目录所有文件，通过正则表达式匹配文件名中的**中文姓名**或**英文花名**。
  3.  **双模式分发**：支持**按小组分发**（文件含小组名则全组成员各得一份）和**按个人分发**两种模式。
  4.  **自动容错**：未能匹配的文件会归入“未匹配文档”文件夹，确保无一遗漏，便于人工复核。

## 如何使用
1.  **安装Python**：确保电脑已安装Python 3.6或以上版本。
2.  **配置名单**：用文本编辑器打开`main.py`，在【第一部分：配置区域】修改`raw_info`变量中的学员名单。
3.  **放置文件**：将所有待分类文件（如`肖三_产品作业.pdf`）放入与脚本相同的目录。
4.  **运行脚本**：双击运行`main.py`，或通过命令行运行 `python main.py`，根据提示选择模式即可。
